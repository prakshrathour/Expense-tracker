# Expense Tracker

This is a simple command-line Expense Tracker built using Python.

It lets you record your daily expenses and view them in a clean format.  
The data is saved in a JSON file, so it stays even after closing the program.

## Features

- Add expenses with category, amount, and note  
- View all expenses in a structured table  
- See total spending  
- View category-wise spending  
- Delete any expense  
- Data persistence using JSON
